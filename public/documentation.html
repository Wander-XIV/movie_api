Movie API

A movie web application

The Movie API will allow users to access information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Query Parameters Request body data format Response body data format
Return a list of All movies to the user /movies GET None None A JSON object holding data about all movies
Return data about a single movie by title to the user /movies/[title] GET :Title None JSON object holding data about the selected movie Ex: 
                    
                      {
                        Title: string,
                        Description: string,
                        Genre: {
                          name: string,
                          description: string,
                        },
                        Director: {
                          Name: string,
                          Birth: string,
                          Death: string,
                          Bio: string,
                        },
                        Actors: string,
                        imgURL: string,
                      }
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genre/[genreName] GET :genreName None A JSON object holding data about the selected genre. Example: 
    
                        
                            {
                                name: 'Thriller', 
                                description: ''
                            }
Get a list of all users /users GET None None A JSON object holding data about all users, structured like:
                        
                            id: 1,
                            username: "J4ne",
                            email: "Jane.Doe@mail.com",
                            password: "vZR34!pj#a",
                            birthday: "06/25/1992",
                            favorites: [],
Allow new users to register /users POST None A JSON object holding data about the user to add. Example: 
                        
                            { 
                                "username": "", 
                                "email": "", 
                                "password": ""
                                "birthday": "", 
                                "favorites": []            
                            }
Allow users to update their user info /users/[username] PUT :Username A JSON object holding the new password. Ex: 
                        
              { 
                "name": "Jane",
                "password": "0000"
              }
JSON object with the updated user. Ex: 
                        
              { 
                id: '1',
                name: 'Jane', 
                password: "0000"
              }
Allow existing users to close account /users/[id] DELETE :id None A message indicating the users account was closed
